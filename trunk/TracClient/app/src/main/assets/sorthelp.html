Sort ticket list

On this screen you can specify the order of sorting.

New sort fields are added default at the end. Use the "^" and "v" buttons on the left to move a field higher or lower in the sorting chain. Use the "^" button on the right to define the sort direction (ascending or descending)
A field can be removed by choosing the trash button.

With the 'Save sort order' button, this sort order defined and applied to the ticket list. With 'Back' changes are not included. and the ticket will list the old sorting.
