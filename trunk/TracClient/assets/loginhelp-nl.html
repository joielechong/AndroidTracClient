Specificeer TRAC server

Met dit scherm kan worden opgegeven welke TRAC server gebruikt dient te worden.
Je kunt de informatie op het apparaat opslaan. Wees ervan bewust dat wachtwoorden worden als platte tekst opgelsagen maar op niet gerootte apparaten is die normaliter niet beschikbaar zijn voor andere apps.
Als opgeslagen informatie beschikbaar is dan wordt een dropdown box getoond om een van de opgeslagen profielen te selecteren.

Als je de fout No peer certificate krijgt, selecteer dan Accepteer Self Signed certificaten. Indien login informatie vereist is dan kan deze ook ingevoerd worden. Gebruik de Verifieer knop om te controleren of de app verbinding kan maken met de server. Met OK wordt de server informatie opgeslagen.

Een paar voorbeeld URLs:

  • TracClient RO: http://van-loon.xs4all.nl/TracClient/rpc
    Geen gebruikersnaam en wachtwoord nodig
  • TracClient RW: https://van-loon.xs4all.nl/TracClient/login/rpc
    Een geldige gebruikersnaam en wachtwoord zijn nodig ook moet voor deze site het certificaat vinkje gezet worden.

De sorteervolgorde word op priority&modified&desc=1 gezet.
Het filter word op max=500&status!=closed gezet.
