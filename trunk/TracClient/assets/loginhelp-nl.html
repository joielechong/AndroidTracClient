Specificeer TRAC server

Met dit scherm kan worden opgegeven welke TRAC server gebruikt dient te worden. Als je de fout No peer certificate krijgt, selecteer dan Accepteer Self Signed certificaten. Indien login informatie vereist is dan kan deze ook ingevoerd worden. Gebruik de Verifieer knop om te controleren of de app verbinding kan maken met de server. Met OK wordt de server informatie opgeslagen.
De sorteervolgorde word op priority&modified&desc=1 gezet.
Het filter word op max=500&status!=closed gezet.
