Detail information

In this screen all information of the ticket is displayed. Only fields with a value are displayed by default, using the menu option Show Empty Fields displays those too.

Individual fields can be modified by selecting with a long-click the field, a popup will be shown to modify its value. When trying to modify one of the following fields 'owner', 'status', 'action', 'resolution' the 'Update status' screen will be presented.
There is a separate menu option to add attachments.

You can also view the attachments if an app is available on your device to handle that file. To view an attachment, select the line of the attachment and Android will ask which app to use as viewer.

When swiping from left to right the previous tocket in the list is selected, from right to left will select the next one (not when changes have not yet been saved).
