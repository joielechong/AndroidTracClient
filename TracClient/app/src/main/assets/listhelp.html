Ticket listing

In this screen an overview of the tickets is shown. On first startup after selecting a TRAC host the selection is all non-closed tickets sorted by priority and modification time. The filtering and sorting can be changed though the menu options Filter and Sort. Use the Search menu option for direct filtering of the ticket list.

If after requesting the list changes appear on the serverthis will be notified in the notiofication bar. Refreshing the overview removes the notification.

The difference between Search and Filter is that the second option is performed by the TRAC server and is remembered over sessions, just like the sort option.
