Version 6.1.0
------------------------------------------------------------------------------------
        - Adding dragging facility for HTML elements - see `myAJAX.draggable` function.
        - Redefine `myXML` object definition into `myAJAX.XML` and `XMLHttp` object 
        definition into `myAJAX.XMLHttp`, while keeping backward compatibility.
        - Adding `$getPosition` method for `HTMLElement`.
        - Adding `$scroll` and `$getMousePosition` methods for `window`.
        - Redesign of `$captureEvent` and `$releaseEvent` methods for `HTMLElement`.
        - Renaming `$style` `HTMLElement`'s method into `$getStyle`.
        - Renaming `$opacity` `HTMLElement`'s method into `$setOpacity`.
        
Version 6.0.3
------------------------------------------------------------------------------------
        - Adding `$releaseEvent` method for `HTMLElement`.
        - Adding `$body` method for `Function`.
        - Redesign of `$captureEvent` method for `HTMLElement`.
        - Very small redesign of `$` method.
        - Bug fixing: javascript code being received through iFrame method is executed twice.

Version 6.0.2
------------------------------------------------------------------------------------
        - In the case when browser does not support `iframe` element and iframe based method is selected,
        an `Error` is thrown: `myAJAX.constants.errorMessages.iframeElementNotSupported`.
        - Adding `$reset` method for `HTMLElement`.
        - Redesign of `$unescapeHTML` method and `myajax.php` file.

Version 6.0.1
------------------------------------------------------------------------------------
        - Adding `myajax.php` file for delivering compressed version of `myajax.js`.

Version 6.0.0
------------------------------------------------------------------------------------
        - Renaming all methods starting with `$`.
        - Modifying `$` function to extend returned objects, with all methods of `HTMLElement`
        and auto assigning an ID if the HTML element doesn't have one already.
        - Redefine prototype methods for `Object` into prototype methods for `HTMLElement`.
        - Modifying `document.getElementById`, `document.createElement`, `document.createTextNode`,
        `document.getElementsByName` and `document.getElementsByTagName` methods, in order
        to return objects with all `HTMLElement`'s methods, inherited.
        - Adding `version` property for `myAJAX` object.
        - Adding `$indexOf` method for `Array` objects.
        - Adding `$show`, `$hide`, `$opacity`, `$remove`, `$toggleOpacity`, `$updateHTML` and 
        `$updateText` methods for `HTMLElement`.
        - Adding `$`, `$extractScripts`, `$stripScripts`, `$escapeHTML` and `$unescapeHTML` methods for 
        `String` objects.
        - Adding `$setProperty` method for `Object`.
        - Adding `document.$createElement' method.
        - Moving `Cookie` object into a separate file `cookie.js`.
        - Other redesigns / optimizations.

Version 5.0.0
------------------------------------------------------------------------------------
        - `myAJAX.urlParameters` can be a string (containing URL parameters or an `id` of a form
          or form element) or an object (pointing to a form or to a single form element).
        - In case that `myAJAX.urlParameters` is referencing a form containing file inputs,
          the form will be automatically submitted through a hidden IFRAME.
        - Adding possibility to send myAJAX's parameters through a single object.
        - Adding `onStartFunction` method for `myAJAX` object.
        - Adding `ajaxMethod` property for the `myAJAX` object.
        - Adding `ID` property for myAJAX object instead of the`getObjectID` method.
        - Adding `enableMultiplyingFileInputs` property for the myAJAX object.
        - Redefine `ajax` function as a statical method for the `myAJAX` object.
        - Renaming `parseObject` method into an `object2QueryString`.
        - Converting `newAJAX` function into an `XMLHttp` object.
        - Adding a `$` function.
        - Adding `toggleDisplay`, `toggleVisibility`, `copyStyle`, `isString`, `isForm`
          and `isFormWithFileInput` methods for `Object`.
        - Adding `getKeys`, `baseName` and `stripCRLF` methods for `String` objects.        
        - Adding `inArray` method for `Array` objects.        
        - Redesign of `base64Encode` and `replaceAll` functions.
        - Adding `utf8Encode`, `utf8Decode`, `base64Decode`, `md5` and `captureEvent` methods
          for `String` objects.
        - Moving all hash functions into a separate JavaScript file `hash.js`
        - Many other redesigns / optimizations.
